LLMCurator
Features Setup

Frontend and Annotation tool
for your LLM

Teams use LLMCurator to annotate data, investigate LLM, and share models.

Key Features

1. Increase privacy:

Track and remove sensitive data from inputs before sending to LLM.

2. Curate data:

Choose and update response from LLM for future training.

3. Work with multiple models:

Investigate and share multiple models in one UI.

4. Easy to setup:

Setup in just 3 commands.

Setup

      
 $ # Install docker from https://docs.docker.com/compose/install/

 $ git clone https://github.com/pushpankar/LLMCurator.io
 $ cd LLMCurator.io
 $ docker-compose up
